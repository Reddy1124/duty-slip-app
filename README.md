# 🚗 Digital Duty Slip System
A Digital Duty Slip Application built to replace manual paper-based trip records used by drivers and transport vendors.

This system enables:
- Real-time trip logging
- Automatic KM and time calculation
- Error reduction through validation
- Instant WhatsApp submission to vendors
- Professional printable duty slip generation

Designed for real-world usage in transport and travel operations.

Built with HTML, CSS, and JavaScript as a lightweight, mobile-first solution.
